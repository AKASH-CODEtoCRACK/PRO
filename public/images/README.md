# Logo Image Folder

## Instructions

**IMPORTANT:** Save your company logo here as `logo.png`

### Steps:
1. Right-click on your logo image
2. Save it as `logo.png`
3. Place it in this folder (`public/images/`)

The logo will then automatically appear in:
- Navbar (top left)
- Footer (company info section)
- Anywhere the Logo component is used

### Recommended Specifications:
- Format: PNG with transparent background
- Minimum size: 200x200 pixels
- Higher resolution is better for quality

Once the logo is saved here, restart your dev server to see it in action!
